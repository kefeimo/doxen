## Overview

This project is a robust development framework built primarily with TypeScript and Next.js, featuring extensive testing infrastructure and multi-environment deployment capabilities. The codebase combines frontend JavaScript/TypeScript development with Rust components, creating a powerful foundation for scalable web applications.

The framework provides developers with a complete toolkit for building, testing, and deploying Next.js applications across multiple environments, with built-in support for various deployment strategies including standalone builds, static exports, and containerized deployments.

## Quick Start

### Environment Variables
The following environment variables are required:
```bash
FIREBASE_PRIVATE_KEY=your_firebase_key
STRIPE_SECRET_KEY=your_stripe_key
POSTGRES_URL=your_postgres_connection_string
TURSO_DB_TOKEN=your_turso_token
CLERK_SECRET_KEY=your_clerk_key
```

### Development Servers
- Next.js applications: http://localhost:3000
- Redis Stack: http://localhost:6379 (Redis) / http://localhost:8001 (Web UI)
- Static export server: http://localhost:8080
- Multi-environment ports: 3001 (dev), 3002 (staging), 3003 (prod)

### Available Scripts
```bash
# Build all packages
npm run build

# Build with native optimizations
npm run build-all

# Clean all dependencies and build artifacts
npm run clean

# Generate new error handling
npm run new-error

# Generate new test files
npm run new-test
```

### Development Mode
```bash
# Start development server
pnpm --filter=next dev
```
