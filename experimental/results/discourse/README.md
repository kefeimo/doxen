# Discourse

A modern forum and community discussion platform built with Ruby on Rails and Ember.js.

## Overview

Discourse is a comprehensive forum software designed for civilized community discussion. It provides a modern, feature-rich platform where communities can engage in meaningful conversations through topics, posts, and real-time interactions. The platform emphasizes user experience with a responsive interface, robust moderation tools, and extensive customization options.

Built for communities of all sizes, Discourse serves as a central hub for discussions, knowledge sharing, and community building. It's used by organizations, open-source projects, hobbyist communities, and businesses looking to foster engagement and provide customer support through a discussion-based format.

## Features

- **Topic and Post Management** - Create, edit, and organize discussions with rich text formatting
- **User Authentication and Profiles** - Comprehensive user management with customizable profiles and badges
- **Advanced Search** - Full-text search across topics, posts, and user content
- **Moderation Tools** - Review system, flagging, and administrative controls for content management
- **Categories and Tags** - Organize content with hierarchical categories and flexible tagging
- **Real-time Notifications** - Live updates and notification system for user engagement
- **Group Management** - Create and manage user groups with custom permissions
- **File Uploads and Media** - Support for images, documents, and secure media handling
- **Theming and Customization** - Custom themes, stylesheets, and JavaScript extensions
- **Email Integration** - Email notifications, webhooks, and mailing list functionality
- **Mobile-Responsive Design** - Optimized experience across desktop and mobile devices
- **API Access** - Comprehensive REST API for integrations and custom applications
- **Plugin System** - Extensible architecture with plugin support
- **Bookmarking and Drafts** - Save content for later and auto-save draft functionality

## Tech Stack

**Backend:**

**Frontend:**
- Ember.js with JavaScript/TypeScript (2,767 files)

**Architecture:**
- Single-page application (SPA) frontend

## API Overview

Discourse provides a comprehensive REST API with 993+ endpoints covering:

- **Content Management** - Topics, posts, categories, and tags
- **User Operations** - Authentication, profiles, groups, and badges  
- **Administrative Functions** - Moderation, site configuration, and analytics
- **Real-time Features** - Notifications, presence, and messaging
- **Media Handling** - File uploads, avatars, and secure content delivery

The API supports both public read operations and authenticated user actions, with role-based permissions for administrative functions.
